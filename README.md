# Conversor de bases
# O conversor de bases converte:
Binario, Octal, e Hexadecimal para Decimal.
Decimais para Binario, Octal, e Hexadecimal.


O codigo foi feito com dois conversores usando estruturas de repetição e listas. Subistitui letras por números ( A=10 ~ F=15 ) no caso do conversor de Hexadecimal.
